# Folder Icon Manager

一个面向 Windows 的轻量文件夹图标替换工具。它将系统属性中的多步操作收敛为：选择目标文件夹、选择 ICO 图标、立即应用。

## 功能

- 选择或拖入目标文件夹。
- 选择或拖入 `.ico` 图标文件。
- 立即应用文件夹自定义图标，并通知资源管理器刷新显示。
- 一键恢复 Windows 默认文件夹图标。
- 收藏常用图标库文件夹：单击 ICO 可预览，双击可直接应用到当前目标文件夹。
- 主操作区选择的 ICO 与收藏栏 ICO 共用预览区。
- 可通过目标路径旁的眼睛图标，在资源管理器中定位目标文件夹。
- 可通过预览区图标，在资源管理器中定位当前 ICO 文件。

## 使用

1. 在“请选择目标文件夹”区域点击浏览，或将一个文件夹拖入虚线框。
2. 在“请选择 ICO 图标”区域点击浏览，或将一个 `.ico` 文件拖入虚线框。
3. 确认路径后点击“立即应用”。状态栏会显示本次应用的 ICO 文件名。
4. 需要还原时，保留目标文件夹路径并点击“恢复默认”。

### 图标收藏栏

1. 点击“添加图标库”，或将包含 ICO 文件的文件夹拖到收藏栏。
2. 展开某个收藏文件夹查看其中的 `.ico` 文件。非 ICO 文件不会显示。
3. 单击图标更新主操作区路径并在左侧预览；双击图标立即应用到当前目标文件夹。
4. 在收藏文件夹上右键选择“从收藏夹移除”可移除记录，不会删除源文件夹或其中的文件。

## 重要限制

- 仅支持本地文件夹与 `.ico` 文件，不支持网络目录。
- 对 `C:` 盘，仅允许修改桌面根目录下的文件夹及其任意层级子文件夹；其他本地磁盘不受此范围限制。
- 工具只在目标文件夹的自定义图标设置中引用 ICO 的原始路径，不复制 ICO 文件。应用后请不要移动、重命名或删除被引用的 ICO，否则图标可能失效。
- 收藏数据保存在 EXE 同目录的 `favorites.json`。删除整个工具目录即可一并清理收藏数据；不会在系统目录或用户配置目录额外写入数据。

## 运行要求

正式发布包为 `win-x64` 自包含版本，适用于 64 位 Windows 10/11。解压后运行 `FolderIconManager.exe` 即可，无需单独安装 .NET 运行时。

## 本地构建

需要 .NET 8 SDK：

```powershell
dotnet build .\FolderIconManager.sln -c Release
dotnet publish .\src\FolderIconManager\FolderIconManager.csproj `
  -c Release -r win-x64 --self-contained true `
  -o .\publish\FolderIconManager-win-x64
```

发布包请通过 GitHub Releases 获取；源码仓库中的 `inbox/`、`project-audit/` 和本地收藏数据均不纳入 Git。
