# Folder Icon Manager - Claude Code 工作指引

## 项目目标

开发一款 Windows 文件夹图标管理工具。

核心能力：

- 为 Windows 文件夹应用 ICO 图标
- 恢复 Windows 默认文件夹图标
- 管理本地 ICO 图标库
- 管理收藏夹与分组
- 支持拖拽操作

## 技术栈

优先采用：

- Tauri v2
- React
- TypeScript
- Rust

目标平台：

- Windows 10
- Windows 11

## 图标处理规则

必须遵守：

- 使用 desktop.ini 实现文件夹图标
- 使用源 ICO 路径引用方案
- 禁止复制 ICO 到目标文件夹
- 禁止删除 desktop.ini
- 禁止覆盖 desktop.ini 中与图标无关的内容

恢复默认图标时：

- 仅移除本工具写入的图标配置
- 保留其他配置
- 即使 desktop.ini 为空，也不允许删除该文件

## 图标库规则

必须遵守：

- 不使用数据库
- 不使用缓存
- 不建立索引
- 每次打开图标库时实时扫描目录
- 只读取 .ico 文件
- 新增或删除 ICO 后自动体现
- 不提供刷新按钮

## 数据存储规则

收藏夹配置：

- 使用 JSON 文件存储

禁止：

- 数据库
- 注册表
